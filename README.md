# Hardware Ecosystem Analysis: MediaTek, ASUS, Corsair, & Kioxia

## 🏢 Project Overview
This group assignment provides a deep-dive analysis of the modern hardware ecosystem. We explore how different hardware components—from processors to storage—interact with Operating Systems and System Programming layers to create a functional computing environment.

## 🔍 Featured Hardware Entities
Our research focuses on the architectural and system-level roles of four industry leaders:

1. MediaTek (Processor Architecture): Analysis of ARM-based SoC designs and instruction set efficiency.
2. ASUS (System Integration): Documentation on motherboard firmware (BIOS/UEFI) and hardware-software communication.
3. Corsair (Memory Systems): Examination of high-speed RAM architecture and volatile memory management.
4. Kioxia (Storage Persistence): Insights into NAND flash technology and non-volatile storage controllers.

## 🛠 Technical Focus
* Instruction Sets: How CPUs execute system calls.
* Memory Management: The relationship between system programming and hardware resources.
* Firmware/Drivers: The bridge between the hardware ecosystem and the Operating System.
